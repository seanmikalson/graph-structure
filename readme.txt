To compile:

Navigate to directory where source files are located and run the following command:

javac *.java

To run program enter the following command:

java Assign4 INPUTFILE QUERYFILE DEPTHFIRSTOUPUT BREADTHFIRSTOUTPUT DIJKSTRAOUTPUT

where INPUTFILE and QUERYFILE are properly formatted text files 
Note: QUERYFILE must not have any extra spaces at the end of file and will cause a crash
